# IronFlow - 조선기자재 생산 자동 스케줄링 시스템

조선기자재 생산 프로세스의 역산 스케줄링을 자동으로 계산하는 웹 애플리케이션입니다.

## 주요 기능

- ✅ **동적 공정 관리**: 공정을 자유롭게 추가/수정/삭제
- ✅ **프로젝트 마스터 등록**: 엑셀 업로드 또는 직접 입력으로 프로젝트 정보 관리
- ✅ **공정별 소요기간 입력**: 각 블록별 공정 소요 일수 관리
- ✅ **역산 스케줄링**: 납기일 기준으로 각 공정의 시작일/종료일 자동 계산
- ✅ **팀별 휴무일 설정**: 각 팀의 근무 요일과 휴무일을 개별 설정
- ✅ **간트 차트 시각화**: 스케줄을 한눈에 확인
- ✅ **엑셀 템플릿 다운로드**: 표준 입력 양식 제공
- ✅ **NumPy 가속 계산**: 대용량 데이터도 빠르게 처리

## 설치 방법

### 1. 저장소 클론

```bash
git clone https://github.com/banlan21-maker/splan.git
cd splan
```

### 2. 가상환경 생성 및 활성화 (권장)

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. 패키지 설치

```bash
pip install -r requirements.txt
```

### 4. 애플리케이션 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 열리며, 기본 주소는 `http://localhost:8501`입니다.

## 사용 방법

### 1. 공정 설정 (Admin)
- 시스템에서 사용할 공정을 정의합니다.
- 공정 유형: `Duration` (기간) 또는 `Milestone` (특정일)
- 공정 순서를 설정하여 스케줄 계산 순서를 결정합니다.

### 2. 기초정보 관리
- **프로젝트 마스터 등록**: 호선번호, 블록번호, 중량, 납기일 등록
  - 엑셀 파일 업로드 또는 직접 입력
  - 표준 입력 양식 다운로드 가능
- **공정별 소요기간 입력**: 각 블록별 공정 소요 일수 입력

### 3. 스케줄링 메인
- 휴무일 설정 (공통 휴무일, 팀별 휴무일)
- 스케줄 계산 실행
- 결과 확인 및 엑셀 다운로드

## 기술 스택

- **Python 3.14+**
- **Streamlit**: 웹 애플리케이션 프레임워크
- **Pandas**: 데이터 처리
- **NumPy**: 고성능 날짜 계산
- **Plotly**: 간트 차트 시각화
- **openpyxl**: 엑셀 파일 처리
